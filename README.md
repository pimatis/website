# Pimatis Website

## Project Overview
Pimatis Website is a modern web application built using SvelteKit and styled with Tailwind CSS.

## Tech Stack
- [SvelteKit](https://kit.svelte.dev/) - Full-stack framework for building web applications
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
- [Vite](https://vitejs.dev/) - Next generation frontend tooling

## Setup and Installation

### Prerequisites
- Node.js (LTS version recommended)
- npm, yarn or bun

### Installation
Clone the repository and install dependencies:

```bash
git clone https://github.com/Pimatis/website.git
cd website
bun install
```

### Development

Run the development server:

```bash
bun run dev
```

Visit `http://localhost:5173` to see the application.
